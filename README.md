# Midterm Lab Exam Submission
Student ID: 22299319

## Features
- View Landmarks on Map
- List all Landmarks
- Create new Landmark with GPS and Photo

## Schema
- **Landmark**: id (int), title (string), lat (double), lon (double), image (url string).

## Known Limitations
- The 'Update' feature is planned for v2.0.
- Map requires a valid API key to render tiles.

## How to Run
1. Clone repo
2. Add Google Maps API Key in AndroidManifest
3. Run on Emulator
